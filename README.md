# 🎓 EduSphere - Online Learning Platform

## 📌 Overview

EduSphere is a responsive Online Learning Platform built using HTML, CSS and JavaScript.

The platform demonstrates modern frontend development concepts with an attractive UI and interactive features.

---

## 🚀 Features

- Responsive Website
- Student Dashboard
- Teacher Dashboard
- Popular Courses
- Progress Tracker
- Quiz System
- Certificate Section
- Contact Form
- Dark Mode
- Scroll to Top
- Mobile Navigation
- Smooth Scrolling

---

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
